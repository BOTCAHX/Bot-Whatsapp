# Install

---
### WINDOWS / VPS / RDP USER
* Download & Install Git [`Git`](https://git-scm.com/downloads)
* Donwload & Install NodeJS [`NodeJS`](https://nodejs.org/en/download)

```bash
git clone https://github.com/rayyreall/Whatsapp_Bot
cd Bot-Whatsapp
npm build
```
